Harry Potter API

A very cool API to get information on the Harry potter novel series/movies.

What is it?

This is a API that provides endpoints for everything in the Harry Potter World. From the Hogwarts Houses to every spell, we've got it all!

Why?

There are many API's for many popular movie/book series, but surprisingly, there isnt much out there for Harry Potter. Everything I found either lacked a lot of information, or were abandoned. This api aims to fix that!

Is it finished?

At the time of writing, no it is not. It is currently in development, and is being updated every day.

Can I contribute?

Of course you can! Head over to https://github.com/YeffyCodeGit/PotterAPI, add your changes, open a pull request and we will review it!
